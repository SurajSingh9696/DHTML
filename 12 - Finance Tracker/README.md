## How to Use

1. **Add a Transaction**:
   - Enter description (e.g., "Monthly Salary")
   - Enter amount
   - Select a category
   - Choose date
   - Select type (Income or Expense)
   - Click "Add Transaction"

2. **View Balances**:
   - Total Balance shows your net worth
   - Total Income displays all earnings
   - Total Expenses shows all spending

3. **Filter Transactions**:
   - Click "All" to see everything
   - Click "Income" to see only income
   - Click "Expenses" to see only expenses

4. **Delete Transaction**:
   - Click the trash icon on any transaction to remove it
