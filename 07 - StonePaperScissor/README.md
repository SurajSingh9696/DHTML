# Stone Paper Scissors

## Description
A modern, interactive Stone Paper Scissors game where you play against the computer. Features real-time score tracking and instant game results with a beautiful gradient UI.

## Features
- **Play Against Computer**: Real-time gameplay with instant decisions
- **Score Tracking**: Keep track of your wins vs computer
- **Emoji-Based Interface**: Visual representation with fun emojis
- **Game Reset**: Start fresh anytime with Reset button
- **Modern Design**: Gradient backgrounds and smooth animations
- **Responsive**: Works seamlessly on all devices
- **Instant Feedback**: See results immediately

## Game Rules
- **Stone** beats **Scissors**
- **Scissors** beats **Paper**
- **Paper** beats **Stone**
- Same choice results in a **Draw**

## Tech Stack
- **HTML5**: Semantic markup
- **CSS3**: Modern styling with gradients and animations
- **JavaScript (Vanilla)**: Pure JS game logic
- **Google Fonts**: Poppins
