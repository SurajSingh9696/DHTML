# Calculator

## Description
A modern, feature-rich calculator with an intuitive dark theme interface. Supports basic arithmetic operations and advanced functions with a clean, responsive design.

## Features
- **Complete Operations**: Addition (+), Subtraction (−), Multiplication (×), Division (÷)
- **Advanced Functions**: Percentage (%), Exponentiation (^)
- **Full Number Support**: Decimal point support with input validation
- **Expression History**: Sub-display shows your calculation history
- **Clear Function**: CE button to reset calculations
- **Overflow Protection**: Handles large numbers gracefully
- **Modern UI**: Dark theme with gradient buttons and smooth animations
- **Responsive Design**: Works perfectly on desktop, tablet, and mobile
- **Keyboard Support**: Press buttons with visual feedback

## Tech Stack
- **HTML5**: Semantic markup
- **CSS3**: Modern styling with gradients and animations
- **JavaScript (Vanilla)**: Pure JS without dependencies
- **Google Fonts**: Roboto Mono and Poppins

## How to Use
1. Open `calindex.html` in a web browser
2. Click number buttons to input values
3. Click operator buttons to select operation
4. Click "=" to calculate the result
5. Click "CE" to clear and start fresh

## Operations
| Symbol | Function | Example |
|--------|----------|---------|
| + | Addition | 5 + 3 = 8 |
| − | Subtraction | 10 − 4 = 6 |
| × | Multiplication | 6 × 7 = 42 |
| ÷ | Division | 20 ÷ 4 = 5 |
| % | Percentage | 50 ÷ 200 % = 25 |
| ^ | Power | 2 ^ 3 = 8 |
