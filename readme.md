# DHTML Projects Collection

A comprehensive collection of modern, responsive web applications built with vanilla HTML5, CSS3, and JavaScript. Each project features a clean, gradient-based UI design with optimized logic and full functionality.

### Health & Wellness
- **BMI Calculator** - Calculate and track Body Mass Index with health category classifications
- **Temperature Converter** - Convert between Celsius, Fahrenheit, and Kelvin scales

### Utilities
- **Calculator** - Scientific calculator with modern dark theme interface
- **Unit Converter** - Convert 100+ units across 8 categories (length, area, volume, mass, data, speed, time, temperature)
- **Currency Converter** - Real-time currency exchange rates powered by API integration
- **Get Any Time** - World clock showing time for any timezone or geographical coordinates

- **Tic Tac Toe (Cris Cross)** - Classic two-player game with modern interface
- **Stone Paper Scissors** - Play against the computer with emoji-based controls

- **Gender Guesser** - Predict gender from name using Genderize API
- **Country Guesser** - Identify country origin of names using Nationalize API

- **ToDo List** - Task management with local storage persistence
- **Weather Forecaster** - Real-time weather and forecasts for any location
