# Get Any Time

## Description
A modern time zone and coordinate-based time display application. Get accurate current time for any timezone or specific geographical location using the TimeAPI.

## Features
- ✅ **Timezone Selection**: Choose from 400+ IANA timezone definitions
- ✅ **Coordinate-Based Time**: Get time for any latitude/longitude location
- ✅ **Real-Time Updates**: Live clock updating every second
- ✅ **Accurate Data**: Powered by TimeAPI for precision
- ✅ **Modern Interface**: Clean gradient design with responsive layout
- ✅ **Instant Results**: No waiting - results appear immediately
- ✅ **Multiple Input Methods**: Both timezone and coordinates supported
- ✅ **Mobile Responsive**: Perfect on all devices

## Tech Stack
- **HTML5**: Semantic markup structure
- **CSS3**: Modern gradients and animations
- **JavaScript (Vanilla)**: Time calculation and API integration
- **TimeAPI**: Accurate time zone and coordinate-based time data
- **Google Fonts**: Poppins typography

## How to Use
1. Open `gtindex.html` in a web browser
2. **For Timezone**:
   - Select a timezone from the dropdown list
   - Click "Get Time" button
   - View the current time for that timezone
3. **For Coordinates**:
   - Enter latitude (e.g., 40.7128)
   - Enter longitude (e.g., -74.0060)
   - Click "Get Time" button
   - View current time for that location

## Files
- `gtindex.html` - Main time display interface
- `gtstyle.css` - Modern responsive styling
- `gtscript.js` - Timezone and time logic

## Browser Support
- Chrome (latest)
- Firefox (latest)
