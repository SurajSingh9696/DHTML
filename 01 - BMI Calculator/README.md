# BMI Calculator

## Description
A modern, responsive web application to calculate your Body Mass Index (BMI) with instant results and visual health category indicators. Get personalized health recommendations based on your BMI value.

## Features
- **Responsive Design**: Optimized for desktop, tablet, and mobile devices
- **Modern UI**: Gradient backgrounds, smooth animations, and polished interface
- **Easy Input**: Intuitive fields for height (meters) and weight (kilograms)
- **Instant Calculation**: Click button or press Enter to calculate
- **Health Categories**: Visual BMI range reference with color-coded categories
- **Input Validation**: Error handling for invalid inputs
- **Accessibility**: Proper labels and semantic HTML structure

## Health Categories
- **Underweight**: BMI < 18.5
- **Healthy Weight**: BMI 18.5 - 24.9
- **Overweight**: BMI 25 - 29.9

## Usage
1. Open `bmiindex.html` in a web browser
2. Enter your height in meters (e.g., 1.75)
3. Enter your weight in kilograms (e.g., 70)
4. Click "Calculate BMI" or press Enter
5. View your BMI value and health status
6. Reference the range chart for BMI categories
