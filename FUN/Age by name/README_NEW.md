# Age Guesser

## Description
Predict your probable age based on your name! This fun web application uses the Agify API to estimate age from names using historical data.

## Features
- **Instant Predictions**: Get age estimates instantly
- **Global Database**: Analyzes millions of names worldwide
- **Confidence Indicator**: Shows confidence level based on data points
- **Modern UI**: Beautiful gradient design with smooth animations
- **Responsive**: Works on all devices
- **Real-time Results**: Displays results immediately after search

## Tech Stack
- **HTML5**: Semantic markup
- **CSS3**: Modern gradients and responsive design
- **JavaScript (Vanilla)**: Fetch API for predictions
- **API**: Agify API
- **Google Fonts**: Poppins

## How to Use
1. Open `aninex.html` in a web browser
2. Enter your name in the input field
3. Click "Predict" or press Enter
4. View your predicted age and confidence level

## API Used
- **Agify API**: https://api.agify.io
- Predicts age based on first names
- Uses global database of names and ages

## Files
- `aninex.html` - Main HTML structure
- `anstyle.css` - Modern responsive styling
- `anscript.js` - Age prediction logic

## Browser Support
- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)

## Notes
- Requires internet connection for API calls
- Confidence percentage is based on sample size
- Results are statistical estimates, not actual ages
