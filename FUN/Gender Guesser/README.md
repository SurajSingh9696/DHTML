# Gender Guesser

## Description
Predict your probable gender based on your name! This fun application uses the Genderize API to estimate gender from names based on historical data and trends.

## Features
- **Instant Predictions**: Get gender predictions instantly
- **Confidence Indicator**: Shows accuracy percentage of prediction
- **Global Database**: Analyzes millions of names worldwide
- **Modern UI**: Beautiful gradient design with animations
- **Responsive**: Works on desktop, tablet, and mobile
- **Real-time Results**: See predictions immediately

## Tech Stack
- **HTML5**: Semantic markup
- **CSS3**: Modern gradients and responsive design
- **JavaScript (Vanilla)**: Fetch API for predictions
- **API**: Genderize API
- **Google Fonts**: Poppins

## How to Use
1. Open `ggindex.html` in a web browser
2. Enter your name in the input field
3. Click "Predict" or press Enter
4. View your predicted gender and confidence level

## API Used
- **Genderize API**: https://api.genderize.io
- Predicts gender based on first names
- Uses global database of names and gender patterns

## Files
- `ggindex.html` - Main HTML structure
- `ggstyle.css` - Modern responsive styling
- `ggscript.js` - Gender prediction logic
