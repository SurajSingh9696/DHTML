# TaskFlow - Advanced Todo List

A modern, feature-rich task management application built with HTML, CSS, and JavaScript.

## Features

### Core Functionality
- ✅ **Add Tasks** - Create tasks with title, category, priority, and due date
- ✅ **Complete Tasks** - Mark tasks as completed with smooth animations
- ✏️ **Edit Tasks** - Modify task details through a modal interface
- 🗑️ **Delete Tasks** - Remove individual tasks or clear all at once
- 🔍 **Search** - Real-time search across task titles and categories
- 📊 **Statistics** - Live stats showing total, active, completed tasks and progress
- 💾 **Local Storage** - All tasks persist between sessions
- 🎯 **Priority Levels** - High, Medium, Low priority with color coding
- 📁 **Categories** - 8 predefined categories (Work, Personal, Shopping, Health, Learning, Finance, Home, Other)
- 📅 **Due Dates** - Track deadlines with overdue indicators
- 🎨 **Theme Toggle** - Switch between dark and light themes
- 📱 **Fully Responsive** - Works seamlessly on all devices

### Filtering & Sorting
- Filter by: All, Active, Completed
- Sort by: Date (Newest/Oldest), Priority, Title (A-Z)
- Real-time search functionality

### Advanced Features
- **Clear Completed** - Remove all completed tasks at once
- **Delete All** - Clear entire task list with confirmation
- **Overdue Indicators** - Visual alerts for overdue tasks
- **Progress Tracking** - Percentage-based completion progress
- **Edit Modal** - Dedicated interface for editing tasks
- **Smooth Animations** - Fade, slide, and pulse effects throughout

## How to Use

1. **Add a Task**
   - Enter task title
   - Select category
   - Choose priority level
   - Set due date
   - Click "Add Task"

2. **Manage Tasks**
   - Click checkbox to mark complete/incomplete
   - Click edit icon to modify task details
   - Click trash icon to delete a task

3. **Filter & Search**
   - Use filter buttons (All/Active/Completed)
   - Type in search box for real-time filtering
   - Change sort order from dropdown

4. **Quick Actions**
   - "Clear Completed" - Remove all finished tasks
   - "Delete All" - Clear entire task list

5. **Theme Toggle**
   - Click sun/moon icon in top-right to switch themes

## Keyboard Shortcuts
- Enter in any form field submits the form
- Escape closes the edit modal
