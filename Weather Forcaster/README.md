# Weather Forecaster

## Description
A comprehensive weather application providing real-time weather data and forecasts for any location worldwide. View current conditions with detailed metrics and visual weather indicators.

## Features
- ✅ **Global Search**: Weather for any city, town, or location
- ✅ **Real-Time Data**: Current temperature and weather conditions
- ✅ **Detailed Metrics**: Temperature, humidity, wind speed, and "feels like" data
- ✅ **Weather Icons**: Visual representation of weather conditions
- ✅ **Location Maps**: Direct link to Google Maps for the searched city
- ✅ **Modern Interface**: Gradient backgrounds with responsive design
- ✅ **Instant Results**: No waiting - results appear immediately
- ✅ **Mobile Friendly**: Fully responsive on all device sizes

## Weather Data Provided
- Current temperature (°C or °F)
- "Feels like" temperature perception
- Humidity percentage
- Wind speed
- Weather condition (Clear, Cloudy, Rainy, Snowy, etc.)
- Weather condition icons
- Location coordinates (Latitude, Longitude)

## Tech Stack
- **HTML5**: Semantic markup structure
- **CSS3**: Modern gradients and responsive layouts
- **JavaScript (Vanilla)**: API integration and DOM manipulation
- **WeatherAPI.com**: Real-time weather data provider
- **Google Fonts**: Poppins typography

## How to Use
1. Open `wfindex.html` in a web browser
2. Enter a city name in the search input field
3. Click "Get Weather" button or press Enter
4. View weather details including:
   - Current temperature and condition
   - Humidity and wind speed
   - "Feels like" temperature
   - Weather icon representation
5. Click map icon to view location on Google Maps
6. Search another location anytime
