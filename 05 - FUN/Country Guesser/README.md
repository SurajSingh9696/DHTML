# Country Guesser

## Description
Discover your country of origin based on your name! This fun application uses the Nationalize API to predict which country your name is most common in.

## Features
- **Instant Predictions**: Get country predictions in seconds
- **Probability Score**: Shows how likely your name is from that country
- **Global Database**: Analyzes billions of names worldwide
- **Modern UI**: Beautiful gradient design with smooth animations
- **Responsive**: Works on all device sizes
- **Real-time Results**: See predictions immediately

## Tech Stack
- **HTML5**: Semantic markup
- **CSS3**: Modern gradients and animations
- **JavaScript (Vanilla)**: Fetch API for predictions
- **API**: Nationalize API
- **Google Fonts**: Poppins

## How to Use
1. Open `cgindex.html` in a web browser
2. Enter your name in the input field
3. Click "Guess" or press Enter
4. View your predicted country and probability percentage

## API Used
- **Nationalize API**: https://api.nationalize.io
- Predicts country based on first names
- Uses global database of names and country distributions

## Files
- `cgindex.html` - Main HTML structure
- `cgstyle.css` - Modern responsive styling
- `cgscript.js` - Country prediction logic

## Browser Support
- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)

## Notes
- Requires internet connection for API calls
